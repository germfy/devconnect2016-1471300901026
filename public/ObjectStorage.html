IBM® Object Storage for Bluemix® proporciona acceso a una cuenta Swift Object Storage completa para gestionar los datos. Swift ofrece una plataforma de almacenamiento completamente distribuida a la que se puede acceder mediante API. Puede utilizarla directamente en sus aplicaciones o para copias de seguridad, ya que es la solución ideal para el almacenamiento escalable y eficaz.

IBM Object Storage for Bluemix utiliza OpenStack Identity (Keystone) para la autenticación y se puede acceder a él directamente utilizando las llamadas de OpenStack Object Storage (Swift) API v1. IBM Object Storage se puede enlazar a una aplicación de Bluemix o se puede acceder desde fuera de una aplicación de Bluemix.

Hay más información y documentación disponibles sobre el uso de OpenStack Swift y Keystone en el sitio de documentación de OpenStack.
